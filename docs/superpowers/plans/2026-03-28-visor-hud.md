# Visor HUD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the basic HUD with an immersive suit visor display — Orbitron sci-fi font, holographic blue-green tint, crosshair with glow, suit O2/power vitals, locked target reticle, warning vignette, and visor degradation.

**Architecture:** New `visor.rs` replaces old `hud.rs` drawing. New `suit.rs` in sa_survival for suit resources. Orbitron font loaded at startup. All visor elements drawn via egui Painter with semi-transparent glass projection aesthetic.

**Tech Stack:** egui Painter API, Orbitron TTF font, existing HUD render pass.

```rust
// crates/sa_survival/src/suit.rs
//! Suit resources: emergency oxygen and power when ship systems fail.

/// Suit survival resources — last line of defense.
#[derive(Debug, Clone)]
pub struct SuitResources {
    /// Oxygen remaining (0.0 to 1.0). Drains when ship O2 unavailable.
    pub oxygen: f32,
    /// Battery power remaining (0.0 to 1.0). Drains when ship power unavailable.
    pub power: f32,
}

/// Suit O2 drain rate per second when ship O2 is unavailable.
const SUIT_O2_DRAIN: f32 = 0.002; // ~8 minutes
/// Suit power drain rate per second when ship power is unavailable.
const SUIT_POWER_DRAIN: f32 = 0.001; // ~16 minutes
/// Suit recharge rate per second when ship resources are available.
const SUIT_RECHARGE: f32 = 0.01; // fast recharge from ship

impl SuitResources {
    pub fn new() -> Self {
        Self { oxygen: 1.0, power: 1.0 }
    }

    /// Update suit based on ship state.
    /// - Ship O2 available: recharge suit O2
    /// - Ship O2 unavailable: drain suit O2
    /// - Ship power available: recharge suit power
    /// - Ship power unavailable: drain suit power
    pub fn update(&mut self, dt: f32, ship_o2_available: bool, ship_power_available: bool) {
        if ship_o2_available {
            self.oxygen = (self.oxygen + SUIT_RECHARGE * dt).min(1.0);
        } else {
            self.oxygen = (self.oxygen - SUIT_O2_DRAIN * dt).max(0.0);
        }

        if ship_power_available {
            self.power = (self.power + SUIT_RECHARGE * dt).min(1.0);
        } else {
            self.power = (self.power - SUIT_POWER_DRAIN * dt).max(0.0);
        }
    }
}

impl Default for SuitResources {
    fn default() -> Self { Self::new() }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn starts_full() {
        let s = SuitResources::new();
        assert_eq!(s.oxygen, 1.0);
        assert_eq!(s.power, 1.0);
    }

    #[test]
    fn drains_when_ship_unavailable() {
        let mut s = SuitResources::new();
        s.update(1.0, false, false);
        assert!(s.oxygen < 1.0);
        assert!(s.power < 1.0);
    }

    #[test]
    fn recharges_when_ship_available() {
        let mut s = SuitResources::new();
        s.oxygen = 0.5;
        s.power = 0.5;
        s.update(1.0, true, true);
        assert!(s.oxygen > 0.5);
        assert!(s.power > 0.5);
    }

    #[test]
    fn o2_lasts_about_8_minutes() {
        let mut s = SuitResources::new();
        s.update(480.0, false, true); // 8 minutes
        assert!(s.oxygen < 0.05, "O2 should be nearly empty after 8min: {}", s.oxygen);
    }

    #[test]
    fn power_lasts_about_16_minutes() {
        let mut s = SuitResources::new();
        s.update(960.0, true, false); // 16 minutes
        assert!(s.power < 0.05, "Power should be nearly empty after 16min: {}", s.power);
    }
}
```

```rust
// crates/spaceaway/src/ui/visor.rs
//! Suit visor HUD: holographic display projected on helmet glass.
//!
//! All drawing uses egui Painter with semi-transparent blue-green tint.
//! Orbitron font for all text. Glow effects via double-draw.

use sa_ship::InteractableKind;

/// Visor color palette.
const VISOR_COLOR: egui::Color32 = egui::Color32::from_rgb(120, 220, 210);
const VISOR_WARN: egui::Color32 = egui::Color32::from_rgb(220, 160, 80);
const VISOR_CRIT: egui::Color32 = egui::Color32::from_rgb(220, 80, 60);

/// All state needed to draw the visor HUD.
pub struct VisorState {
    pub screen_width: f32,
    pub screen_height: f32,
    pub font_scale: f32,
    pub cursor_grabbed: bool,
    pub hovered_kind: Option<InteractableKind>,
    // Suit
    pub suit_o2: f32,
    pub suit_power: f32,
    // Target
    pub target_screen_pos: Option<[f32; 2]>, // None = no target or behind camera
    pub target_off_screen_angle: Option<f32>, // angle for edge chevron when off-screen
    pub target_name: Option<String>,
    pub target_distance_ly: Option<f64>,
    // Time for animations
    pub time: f32,
}

/// Visor font family name (must match what was loaded in UiSystem).
pub const VISOR_FONT: &str = "orbitron";

/// Draw the complete visor HUD.
pub fn draw_visor(ctx: &egui::Context, state: &VisorState) {
    let w = state.screen_width;
    let h = state.screen_height;
    let s = state.font_scale;
    let center = egui::pos2(w / 2.0, h / 2.0);

    // Visor degradation: compute flicker and jitter
    let (flicker_visible, jitter_x, jitter_y, visor_alpha_mult) = compute_degradation(state);

    if !flicker_visible { return; } // visor completely off this frame

    egui::Area::new(egui::Id::new("visor_hud"))
        .fixed_pos(egui::pos2(0.0, 0.0))
        .interactable(false)
        .show(ctx, |ui| {
            let painter = ui.painter();

            // --- Crosshair ---
            if state.cursor_grabbed {
                draw_crosshair(painter, center, s, visor_alpha_mult, jitter_x, jitter_y, &state.hovered_kind);
            }

            // --- Suit O2 (bottom-left) ---
            draw_suit_vital(
                painter, "O2", state.suit_o2,
                egui::pos2(40.0 * s + jitter_x, h - 45.0 * s + jitter_y),
                s, visor_alpha_mult, true,
            );

            // --- Suit Power (bottom-right) ---
            draw_suit_vital(
                painter, "PWR", state.suit_power,
                egui::pos2(w - 40.0 * s + jitter_x, h - 45.0 * s + jitter_y),
                s, visor_alpha_mult, false,
            );

            // --- Locked target reticle ---
            if let Some([tx, ty]) = state.target_screen_pos {
                draw_target_reticle(
                    painter,
                    egui::pos2(tx + jitter_x, ty + jitter_y),
                    state.target_name.as_deref(),
                    state.target_distance_ly,
                    s, visor_alpha_mult,
                );
            } else if let Some(angle) = state.target_off_screen_angle {
                draw_edge_chevron(painter, center, angle, w, h, s, visor_alpha_mult);
            }

            // --- Warning vignette ---
            let min_vital = state.suit_o2.min(state.suit_power);
            if min_vital < 0.8 {
                draw_warning_vignette(painter, w, h, min_vital);
            }
        });
}
```

Then implement the individual draw functions:

**draw_crosshair**: thin ring with center dot, glow via double-draw, color changes based on hovered_kind.

**draw_suit_vital**: text like `O2 98%` in Orbitron font. Alpha based on value (dim when full, bright when draining). Color shifts from visor blue → amber → red.

**draw_target_reticle**: thin ring at target position, name + distance text below. All in visor color with glow.

**draw_edge_chevron**: small `>` rotated to point at the target, drawn at the screen edge.

**draw_warning_vignette**: red-tinted screen edge rects with alpha based on how critical.

**compute_degradation**: returns (flicker_visible, jitter_x, jitter_y, alpha_multiplier) based on suit_power.

All functions use the visor font via `egui::FontId::new(size, egui::FontFamily::Name(VISOR_FONT.into()))`.

- [ ] **Step 2: Load Orbitron font in UiSystem**

```rust
// Load Orbitron font for visor HUD
let mut fonts = egui::FontDefinitions::default();
if let Ok(font_data) = std::fs::read("resources/fonts/Orbitron-Regular.ttf") {
    fonts.font_data.insert(
        "orbitron".to_string(),
        egui::FontData::from_owned(font_data).into(),
    );
    fonts.families.entry(egui::FontFamily::Name("orbitron".into()))
        .or_default()
        .push("orbitron".to_string());
    // Also add as fallback to proportional so it works with FontId
    fonts.families.entry(egui::FontFamily::Proportional)
        .or_default()
        .insert(0, "orbitron".to_string());
    log::info!("Loaded Orbitron font for visor HUD");
} else {
    log::warn!("Orbitron font not found — using default");
}
egui_ctx.set_fonts(fonts);
```
